CSMART Documentation

CSMART is a tool for creating Cougaar societies, configuring the society to form an "experiment", running the experiment using the AppServer, and for monitoring and analysis of results.

See http://www.cougaar.org for an overview of Cougaar and the " Cougaar Developers Guide". Also see the "CSMART User Documentation" for a full list of CSMART features.

See the README for starting-point documentation.

For changes in this release, see the Release Notes.

The Installation and Testing HOW-TO is a step-by-step guide for the installation and testing of CSMART and Cougaar. It is a good starting point for new users.

Additional documentation is provided for Troubleshooting and Known issues. Also see our bug tracking facility.

If you are upgrading from an 8.6.2 install, see the note on upgrading your configuration database.
