Scalability Society

The Scalability Society in CSMART is the scalability module used at the last ALP workshop to measure the scalability of the Cougaar architecture. It allows you to create societies of a rigidly controlled structure, and collect run-time statistics on the running of the society. The specific collected metrics are detailed elsewhere. For details on the purpose of the scalability society, see scalability.

See Scalability Society Help for more information on configuring the society.
