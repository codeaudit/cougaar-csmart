A Cyber impact effects a specific agent. The Cyber attack starts the specified start time and continues for the duration specified.

Target

 The target is the fully qualified agent name that is the target of the cyber attack.

Time

 Time, in milliseconds, when the cyber event starts.

Type

 Type is the type of cyber attack. Currently, there are 4 defined cyber impacts: "DoS Node", "DoS Network", "Kill Node" and "Isolate Node".

Intensity

 Intensity defines the intensity of the attack, the larger the number, the more severe the attack.

Duration

 Duration is the length of the cyber attack.
