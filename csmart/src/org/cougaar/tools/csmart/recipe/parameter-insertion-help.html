Parameter Insertion Recipe

Parameter Insertion Recipes allow users the ability to add a parameter to one or more plugins (or other components), specified by Name.
A Parameter Insertion Recipe contains the following parameters:

Component Fields

Name Type Range
Agent Query Selection List
Plugin Name String
Parameter String


Parameters can be added to any component (that accepts parameters) in any Agent. Creating a Parameter Insertion is an easy way to quickly modify your Agent configuration.

Agent Query

 Query String for Agents in which to find components to modify.

Plugin Name

 Complete Name of the Plugin (or other component). This could be the full class of the Plugin, or <plugin class>|<first argument to plugin>, or other custom component name if you named it yourself. Note that _all_ matching components will get this additional parameter.

Parameter

 New Parameter to add as the last parameter to all matching components.
