Cyber Impact Configuration Help

  • Target [String, agent name]
  • Time [String, milliseconds]
  • Type [String, see below]
  • Intensity [Double, 0 - ?]
  • Duration [Long, 0 - ?]
A Cyber impact effects a specific agent specified in 'Target'. A 'Time' is specified to indicate when the cyber attack starts. There are x 'Types' of cyber attacks defined below that can be used to impact a society. Once a impact is started, it continues for the lenght of the 'Duration', and impacts the society with the specified 'Intensity'.
