Society Monitor Society View

The Society Monitor's Agents View displays the agents in a running Cougaar society. Each agent is labeled with its nomenclature. It is linked to every other agent with which it has a relationship. Agents are laid out top-to-bottom. You may produce new graphs showing only selected relationships, from the File menu. Alternatively, given a selected agent, you can highlight those agents with which that agent has a given relationship (as pictured below).
As with all graph views, a Legend indicates the color scheme. Additionally, each node has a set of attributes of the organization available (by double-clicking or from the View menu when an agent is selected).

Society View Menus
These are menus specific to the Society View. For information on menus and functionality common to all Society Monitor graphs, see the Society Monitor Help menu.

  • File: Relationships...: Display relationships for the selected nodes in a new window. Select the relationship, such as Superior/Subordinate or Customer/Provider from a pull-right menu. To see all the relationships for the society, first select all the nodes (using the Select–>Select All menu item).
  • Show: Show a specific relationship (selected from the Show menu) for the selected nodes. This is most useful for displaying relationships for a single selected node. The related nodes are highlighted in yellow (for Subordinates and Customers) or blue (for Superiors and Providers).
  • Show: Clear: Clear the blue and/or yellow highlighting used to display relationships.
