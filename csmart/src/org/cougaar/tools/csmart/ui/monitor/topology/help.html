Topology Table

This table displays the host, node, agent, and incarnation number for each agent in the society. The incarnation number is the number of times that the node has been started (or restarted); it is initially one. Additional information supplied by the agent is shown in additional columns.

Sorting

The table columns can be sorted alphabetically or numerically by clicking the left mouse button on the column header. For reverse ordering, hold the Shift key.

Column Sizing

The table columns can be resized by positioning the mouse over the border between two columns in the column header, holding the left mouse button, and dragging the border to the new location.

Column Reordering

The table columns can be reordered by positioning the mouse over a column header, holding the left mouse button, and dragging the column to a new location.

Topology Tree

The tree also displays the host, node, and agent. The incarnation number and any additional information from the agent is displayed as a tool tip on the agent node in the tree.

Menus
These are menus specific to the Topology View.

  • File: Close - Close the window.
  • View: Refresh - Contact the society to obtain the current locations of agents and display the new information in the table and tree. The "Refresh" button at the bottom of the display does the same thing.
  • Help: Help - Display this document.
  • Help: About CSMART - Display general information about CSMART.
